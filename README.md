# pomodoro thing

so basically this is a pomodoro timer i made. nothing fancy just gets the job done

---

## what it does

- timer (25/5/15 mins for pomo, short break, long break)
- session dots so u know how many rounds u did
- task list to write down what ur working on
- youtube embed thing — paste a link and the video loads right there, good for watching lectures while timing urself
- notes section (saves automatically so it doesnt disappear when u refresh)
- lil toast popups when a session ends

---

## how to use

just download the 3 files and open index.html in a browser. thats literally it

```
index.html
style.css
script.js
```

keep all 3 in the same folder or the css and js wont load lol

---

## the youtube thing

paste any of these and it works:
- full link like `https://www.youtube.com/watch?v=dQw4w9WgXcQ`
- short link like `https://youtu.be/dQw4w9WgXcQ`
- or just the video id directly `dQw4w9WgXcQ`

press enter or click load. click clear to remove it
